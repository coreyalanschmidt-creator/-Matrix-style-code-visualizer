Raining Code - Lightweight

Use this version if the larger .exe files are blocked by Windows.

Fastest install:
1. Double-click "Install Raining Code.cmd"
2. Press Ctrl+Alt+M anytime to show or hide the effect
3. Press Ctrl+Alt+Q to quit the background helper

What it installs:
- A lightweight fullscreen raining-code visualizer
- A small background hotkey helper
- Startup support so the hotkey stays ready after login
- A desktop shortcut named "Raining Code"
- A desktop shortcut named "Raining Code Settings"

Files:
- Install Raining Code.cmd
- Launch Raining Code.cmd
- Raining Code Settings.cmd
- Raining Code Settings.ps1
- Raining Code Icon.ico
- Raining Code Settings Icon.ico
- Open Raining Code Settings.cmd
- Uninstall Raining Code.cmd
- Raining Code Hotkeys.ps1
- hotkey-settings.json

Default hotkeys:
- Toggle effect: Ctrl+Alt+M
- Quit helper: Ctrl+Alt+Q

Extra keys while the effect is open:
- C changes color
- S changes size
- Space pauses or resumes
- Esc closes the fullscreen effect

Notes:
- The effect starts in bright red mode
- The effect has no on-screen settings overlay
- The settings screen uses tabs for General, Look, and Hotkeys
- The Look tab includes more colors and four sizes: Small, Medium, Large, and Jumbo
- The desktop shortcuts use separate icons for the main app and settings
- Use "Raining Code Settings" to change startup behavior, colors, size, tray visibility, and hotkeys without putting controls on the screen
